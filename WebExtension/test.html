Web Extension Deepfake Detection Test

This page contains test images to verify the web extension is working correctly.

Test Images

Load the web extension and click "Start Detection" to see highlighted images with "Run Deepfake Analysis" buttons.

Instructions

  1. Make sure your Django backend is running on http://127.0.0.1:8000
  2. Load the web extension in Chrome
  3. Click the extension icon and then "Start Detection"
  4. You should see green boxes around the images above
  5. Click "Run Deepfake Analysis" on any image to test the API call
  6. Clipboard Test: Right-click on any image below, select "Copy image", then open the extension popup and press Ctrl+V or click "Paste Image"
